# 🎨 NFT Preview Card

A clean and professional NFT preview card inspired by **NFT marketplaces** like OpenSea.

---

## 🚀 Features
- Dark, modern UI
- Hover overlay with "View NFT" effect
- NFT title, description, price, and timer
- Creator info with avatar

---
